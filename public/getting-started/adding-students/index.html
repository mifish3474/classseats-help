Adding students to a class

Once you’ve created a class, you can begin adding students. In ClassSeats, you can add students one at a time or paste a whole roster using Bulk Add.

Step 1: Go to the Students tab and select your class

  1. Open the Students tab.
  2. Under Select Class Period, click the class you want to work on (for example, “Bell 2”).

Even though the UI says Class Period, think of it as “your class.” You’re simply choosing which class roster you’re editing.

Option A: Single Add (one student at a time)

Single Add is great when:

  • you’re adding a few students
  • your roster changes during the first week
  • you don’t have a clean list ready to paste

To add a student:

  1. Make sure you’re in Single Add (you’ll see the link Switch to Bulk Add).
  2. Type the student name into Student name. Enter names the way you would address students in person: First Last (for example, “Sarah Jones”).
  3. Choose a gender option from the dropdown.
  4. Click + Add.

Repeat for the next student.

Option B: Bulk Add (paste a whole roster)

Bulk Add is best when:

  • you already have a roster list
  • you want to set up a class quickly

To use Bulk Add:

  1. Click Switch to Bulk Add.
  2. Enter one student per line in this format:
    Name, Gender (F/M/O)
  3. Click Add All Students.

Example:

  • Alice Johnson, F
  • Bob Smith, M
  • Charlie Davis, O

Tips for cleaner rosters

  • Keep it consistent. Use the same naming style for everyone (for example, “First Last”).
  • Don’t stress perfection. You can edit student names later.

What happens next

After students are added, most teachers do one of the following next:

  • Create a seating arrangement
  • Take attendance
  • Start quick notes during class
Tip If you already have a roster list, Bulk Add is the fastest way to get set up. If you’re building classes gradually, Single Add works great.
