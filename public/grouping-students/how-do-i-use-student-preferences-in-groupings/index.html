How do I use student preferences in groupings?

When student preferences are set up, ClassSeats uses them automatically whenever you create groups. You don’t need to turn them on or select them — they’re always applied unless you explicitly choose to randomize groups.

Before you start

This works only if student preferences have already been added for your class.

How preferences are applied

  • Honors student preferences
  • Considers grouping history
  • Uses your selected grouping options (such as gender balance)

For step-by-step instructions on creating groups, see: Grouping Students.

Ignore preferences

If you want groups with no preferences or history applied:

  1. Go to the Groups tab.
  2. Check Randomize Groups (ignore all preferences and history).
  3. Click Generate Groups.

When Randomize Groups is checked, ClassSeats ignores:

  • student preferences
  • grouping history
Tip: Randomize Groups is useful for quick activities or icebreakers when you don’t want preferences or past groupings to factor in.
