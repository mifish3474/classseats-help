How do I select preferred seats in my seating arrangement?

Use this when you want to mark specific seats as preferred so you can use them later for preferential seating.

Select preferred seats

  1. Go to the Seating Arrangement Manager.
  2. Find the seating arrangement you want to update and click Edit.
  3. In the editor toolbar, under VIEW, check ✨ Preferred Seats.
  4. Click a seat’s ⭐ to toggle it:
    Outline star = not preferred
    Filled (orange) star = preferred
  5. Repeat for any seats you want to mark.
  6. Click Done to close the editor and save the arrangement.

Unselect a preferred seat

  1. Open the seating arrangement and check ✨ Preferred Seats.
  2. Click the filled ⭐ again to turn it back into an outline.
  3. Click Done to save and close.
Tip: If you don’t see any stars, make sure ✨ Preferred Seats is checked under VIEW.
