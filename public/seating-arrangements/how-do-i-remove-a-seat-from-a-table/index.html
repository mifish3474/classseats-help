How do I remove a seat from a table?

Use this when you want to remove an extra seat from a table for a specific class period.

Remove a seat for one class period

  1. Go to the Groups tab.
  2. Click View Seating Chart.
  3. Find the table you want to remove a seat from.
  4. Click the table’s Table Actions button (top-right corner of the table) and select Seat Editor.
  5. Click the trash can icon on the seat you want to remove.
  6. Confirm that you want to remove the seat.
Tip: To remove seats permanently (for all classes), edit your seating arrangement in the Room Setup tab instead.
