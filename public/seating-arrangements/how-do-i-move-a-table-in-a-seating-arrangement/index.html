How do I move a table in a seating arrangement?

Use this when you want to move a table in your seating arrangement template. This is a global change that affects every class using that arrangement.

Move a table in the arrangement template

  1. Go to the Room Setup tab.
  2. Create a new seating arrangement, or click Edit on an existing arrangement.
  3. Click the table you want to move to select it.
  4. Move the table using the move controls at the top of the editor, or use the arrow keys on your keyboard.
  5. Click Done to close and save the seating arrangement.
Tip: If tables keep bumping into each other while you move them, turn on Disable Collision Rules so you can move tables through each other.
