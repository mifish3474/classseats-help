How do I add a seat to a table?

Use this when you need one more seat at a table for a specific class period.

Add a seat for one class period

  1. Go to the Groups tab.
  2. Click View Seating Chart.
  3. Find the table you want to add a seat to.
  4. Click the table’s Table Actions button (top-right corner of the table) and select Seat Editor.
  5. Click Add Seat.
  6. (Optional) Enter a seat number.
  7. Click Add Seat to confirm.
Tip: To add seats permanently (for all classes), edit your seating arrangement in the Room Setup tab instead.
